# 라우팅 및 페이지 구조

## 라우트 맵 (현재 상태)

```
/ → Home (히어로 캐러셀, 학습경로, 워크플로우, 기능, 통계, CTA)
/intro → IntroPage (소개, FAQ)
/openclaw-guide → OpenClawGuide (8섹션 전문 강의)
/resources → Resources (4카테고리 16주제 + 프롬프트 링크)
/prompt-practice → PromptPractice (기본/고급/실습/퀴즈/갤러리)
/prompt-gallery → PromptGallery (8분야 40개 프롬프트)
/glossary → Glossary (100개 용어사전)
/roadmap → Roadmap (4트랙 학습 로드맵)
/ai-news → AINews (14개 뉴스 + 아카이브)
/community/board → Board (게시판 목록)
/community/board/:id → BoardDetail (게시글 상세)
/community/board/write → BoardWrite (글쓰기, 로그인 필요)
/login → Login
/register → Register
/forgot-password → ForgotPassword
/admin/* → AdminDashboard (관리자 전용)
```

## 컴포넌트 구조

### 레이아웃 (src/components/layout/)
- `Navbar.jsx` - 내비게이션 바 (glassmorphism, 반응형 메뉴)
- `Footer.jsx` - 푸터 (3-column 레이아웃)

### 공통 컴포넌트 (src/components/)
- `HeroCarousel.jsx` - 히어로 캐러셀 (5개 슬라이드, fade 애니메이션)
- `HeroBackground.jsx` - 히어로 배경 (4가지 타입)
- `SEOHead.jsx` - SEO 메타태그 관리
- `Pagination.jsx` - 페이지네이션 (용어사전 5개, 뉴스 10개 단위)
- `CTA.jsx` - Call-to-Action 섹션
- `AuthGuard.jsx` - 인증 가드 (로그인 필요 페이지)
- `AdminGuard.jsx` - 관리자 가드 (admin 전용)

### 레이아웃 래퍼 (src/layouts/)
- `PublicLayout.jsx` - 공개 페이지 레이아웃 (Navbar + Footer 포함)

## 라우팅 특징
- `react-router-dom 7` 사용
- `lazy()` 함수를 활용한 코드 스플리팅
- `AuthGuard`로 로그인 필요 페이지 보호 (글쓰기 등)
- `AdminGuard`로 관리자 전용 페이지 보호
- SPA 리다이렉트: `public/404.html`로 GitHub Pages 새로고침 대응
