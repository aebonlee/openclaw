# OpenClaw - AI 학습 플랫폼 프로젝트 개요

## 기본 정보

| 항목 | 내용 |
|------|------|
| **프로젝트명** | OpenClaw - AI 학습 플랫폼 |
| **URL** | https://openclaw.dreamitbiz.com |
| **GitHub** | https://github.com/aebonlee/openclaw |
| **기반 프로젝트** | Teaching AI (https://teaching.dreamitbiz.com) |

## 목적

AI, 프로그래밍, 데이터 사이언스를 배우는 종합 학습 플랫폼으로, 학습자에게 체계적인 교육 콘텐츠와 AI 기반 교수학습 도구를 제공한다.

## 대상 사용자

- **학생**: AI 및 프로그래밍을 학습하고자 하는 학습자
- **교육자**: AI 도구를 활용한 교수학습 설계가 필요한 교수자
- **AI 학습자**: AI 역량을 체계적으로 강화하고자 하는 일반 사용자

## 주요 기능

### 1. 학습 경로 (6개)

| 학습 경로 | 설명 |
|-----------|------|
| AI 기초 | 인공지능의 기본 개념과 원리 학습 |
| 프롬프트 엔지니어링 | AI 모델과 효과적으로 소통하는 기법 |
| 웹 개발 | HTML/CSS, JavaScript, React 등 웹 기술 |
| 데이터 사이언스 | 데이터 분석, 시각화, 통계 학습 |
| Python | 파이썬 프로그래밍 기초부터 심화 |
| AI 도구 활용 | 실무에서 AI 도구를 활용하는 방법 |

### 2. AI 도구 (5개)

| 도구 | 기능 |
|------|------|
| 강의계획서 생성기 | AI 기반 강의계획서 자동 생성 |
| 루브릭 생성기 | 평가 기준표(루브릭) 자동 생성 |
| 과제 생성기 | 학습 과제 자동 생성 |
| 피드백 생성기 | 학생 과제에 대한 피드백 자동 생성 |
| 학생평가 생성기 | 학생 역량 평가 보고서 자동 생성 |

### 3. 학습 콘텐츠

- **42개 이상의 학습 콘텐츠**: 7개 카테고리로 구성
- **AI 역량 체크리스트**: 학생/교수자별 자가진단 도구
- **프롬프트 엔지니어링 실습**: 기본/고급 기법 연습 환경
- **학습 자료 라이브러리**: 주제별 정리된 학습 자료 모음
- **커뮤니티 게시판**: 학습자 간 소통 공간

## 기반 프로젝트: Teaching AI

OpenClaw는 Teaching AI(teaching.dreamitbiz.com) 프로젝트를 기반으로 한다.

- **동일한 디자인 시스템** 사용 (CSS, 컴포넌트 구조)
- **동일한 Supabase DB** 공유
- **동일한 인증 시스템** 사용
- **차별점**: OpenClaw는 AI 학습에 초점, Teaching AI는 교수학습 도구에 초점
