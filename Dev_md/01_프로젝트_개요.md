# OpenClaw 프로젝트 개요

## 기본 정보
- **프로젝트명**: OpenClaw - AI 학습 플랫폼
- **URL**: https://openclaw.dreamitbiz.com
- **GitHub**: https://github.com/aebonlee/openclaw
- **도메인**: openclaw.dreamitbiz.com (GitHub Pages + CNAME)
- **개발 기간**: 2026-03-28 ~
- **기반 프로젝트**: Teaching AI (teaching.dreamitbiz.com) - 동일 디자인 시스템 공유

## 프로젝트 목적
AI, 프로그래밍, 데이터 사이언스를 체계적으로 학습할 수 있는 종합 무료 학습 플랫폼

## 대상 사용자
- AI 학습자 (입문~중급)
- 프로그래밍 입문자
- 교육자 (OpenClaw 강의 활용)

## 주요 기능 (현재 구현 완료)

### 상단 메뉴 구성 (8개)
1. **소개** (/intro) - 플랫폼 소개, FAQ
2. **OpenClaw 강의** (/openclaw-guide) - 8섹션 전문 교육 콘텐츠
3. **AI 학습자료** (/resources) - AI이론/프로그래밍/데이터분석/AI활용 + 프롬프트
4. **AI 용어사전** (/glossary) - 100개 용어, 5개 카테고리, 검색, 페이지네이션
5. **학습 로드맵** (/roadmap) - 4개 트랙 (AI/ML, 데이터사이언스, 웹개발, 프롬프트)
6. **AI 트렌드** (/ai-news) - 14개 뉴스 카드 + 아카이브 목록 + 페이지네이션
7. **커뮤니티** (/community/board) - 게시판, 댓글, 카테고리 필터
8. 로그인/회원가입

### 핵심 특징
- 5가지 컬러 테마 + 다크/라이트/자동 모드
- 한국어/영어 다국어 지원
- 데스크탑/태블릿/모바일 4단계 반응형
- Glassmorphism 디자인 시스템
- Supabase Auth (Google, Kakao OAuth)
- Teaching AI와 동일 디자인/DB 공유
