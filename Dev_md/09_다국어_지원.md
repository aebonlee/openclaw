# 다국어 지원 (i18n)

## 개요

OpenClaw는 한국어(ko)와 영어(en) 두 가지 언어를 지원한다. 쿠키 기반으로 언어 설정을 유지하며, `useLanguage` 훅의 `t()` 함수를 통해 번역 키에 접근한다.

## 언어 전환

| 항목 | 설명 |
|------|------|
| **지원 언어** | 한국어 (ko), 영어 (en) |
| **기본 언어** | 한국어 (ko) |
| **저장 방식** | Cookie (`lang`) |
| **전환 방법** | Navbar 언어 토글 버튼 |

## Cookie 기반 언어 영속화

```javascript
// 쿠키에서 언어 읽기
function getLanguageFromCookie() {
  const match = document.cookie.match(/(?:^|;\s*)lang=([^;]*)/);
  return match ? match[1] : 'ko'; // 기본값: 한국어
}

// 쿠키에 언어 저장 (1년 유효)
function setLanguageCookie(lang) {
  const expires = new Date();
  expires.setFullYear(expires.getFullYear() + 1);
  document.cookie = `lang=${lang};expires=${expires.toUTCString()};path=/`;
}
```

## LanguageProvider

```jsx
// contexts/LanguageProvider.jsx
import { createContext, useState, useCallback, useMemo } from 'react';
import ko from '../i18n/ko';
import en from '../i18n/en';

const translations = { ko, en };

export const LanguageContext = createContext();

export function LanguageProvider({ children }) {
  const [language, setLanguageState] = useState(getLanguageFromCookie());

  const setLanguage = useCallback((lang) => {
    setLanguageState(lang);
    setLanguageCookie(lang);
  }, []);

  // dot-notation 키 접근
  const t = useCallback((key) => {
    const keys = key.split('.');
    let value = translations[language];
    for (const k of keys) {
      value = value?.[k];
    }
    return value || key; // 번역이 없으면 키 자체 반환
  }, [language]);

  const value = useMemo(() => ({
    language,
    setLanguage,
    t,
  }), [language, setLanguage, t]);

  return (
    <LanguageContext.Provider value={value}>
      {children}
    </LanguageContext.Provider>
  );
}
```

## useLanguage Hook

```javascript
// hooks/useLanguage.js
import { useContext } from 'react';
import { LanguageContext } from '../contexts/LanguageProvider';

export function useLanguage() {
  const context = useContext(LanguageContext);
  if (!context) {
    throw new Error('useLanguage must be used within a LanguageProvider');
  }
  return context;
}
```

## 사용 방법

```jsx
import { useLanguage } from '../hooks/useLanguage';

function MyComponent() {
  const { t, language, setLanguage } = useLanguage();

  return (
    <div>
      <h1>{t('nav.home')}</h1>
      <p>{t('hero.title')}</p>

      {/* 언어 전환 */}
      <button onClick={() => setLanguage(language === 'ko' ? 'en' : 'ko')}>
        {language === 'ko' ? 'EN' : '한국어'}
      </button>
    </div>
  );
}
```

## Dot-notation 키 접근

```javascript
// 예시
t('nav.home')           // "홈" (ko) / "Home" (en)
t('hero.title')         // "AI 학습 플랫폼" (ko) / "AI Learning Platform" (en)
t('tools.syllabus')     // "강의계획서" (ko) / "Syllabus" (en)
t('common.save')        // "저장" (ko) / "Save" (en)
```

## 번역 키 구조

### 전체 키 카테고리

| 카테고리 | 설명 | 예시 키 |
|----------|------|---------|
| `nav` | 내비게이션 | `nav.home`, `nav.intro`, `nav.tools` |
| `hero` | 히어로 섹션 | `hero.title`, `hero.subtitle`, `hero.cta` |
| `tools` | AI 도구 | `tools.syllabus`, `tools.rubric`, `tools.generate` |
| `features` | 기능 소개 | `features.title`, `features.ai`, `features.learning` |
| `stats` | 통계 | `stats.courses`, `stats.tools`, `stats.users` |
| `home` | 홈 페이지 | `home.welcome`, `home.description` |
| `cta` | CTA 섹션 | `cta.title`, `cta.description`, `cta.button` |
| `dashboard` | 대시보드 | `dashboard.title`, `dashboard.recent`, `dashboard.stats` |
| `ai` | AI 관련 | `ai.provider`, `ai.model`, `ai.streaming` |
| `export` | 내보내기 | `export.pdf`, `export.markdown`, `export.json` |
| `settings` | 설정 | `settings.title`, `settings.apiKeys`, `settings.theme` |
| `auth` | 인증 | `auth.login`, `auth.register`, `auth.logout` |
| `community` | 커뮤니티 | `community.board`, `community.write`, `community.comment` |
| `resources` | 학습 자료 | `resources.title`, `resources.category` |
| `intro` | 소개 페이지 | `intro.title`, `intro.mission`, `intro.vision` |
| `eduhub` | 학습 허브 | `eduhub.title`, `eduhub.categories`, `eduhub.courses` |
| `apiGuide` | API 가이드 | `apiGuide.title`, `apiGuide.setup` |
| `footer` | 푸터 | `footer.copyright`, `footer.links`, `footer.familySite` |
| `common` | 공통 | `common.save`, `common.cancel`, `common.delete`, `common.loading` |
