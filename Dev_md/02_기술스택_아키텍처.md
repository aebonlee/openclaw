# 기술 스택 및 아키텍처

## 기술 스택 요약

| 영역 | 기술 |
|------|------|
| **Frontend** | React 19, Vite 6, react-router-dom 7 |
| **Backend/DB** | Supabase (PostgreSQL, Auth, RLS) |
| **Styling** | CSS Custom Properties, Glassmorphism |
| **Font** | Noto Sans KR (Google Fonts), Font Awesome 6.5.1 |
| **Deploy** | GitHub Pages (gh-pages 브랜치) |
| **CI/CD** | GitHub Actions (deploy.yml) |
| **Dev Server** | Port 5175 |

## Frontend

### React 19

- 최신 React 19 사용
- Hooks 기반 함수형 컴포넌트
- Suspense를 활용한 lazy loading
- Context API로 전역 상태 관리 (Theme, Language, Auth, Toast)

### Vite 6

- 빠른 HMR (Hot Module Replacement)
- 최적화된 빌드 출력
- 개발 서버 포트: **5175**

### react-router-dom 7

- 선언적 라우팅
- lazy() 함수를 사용한 코드 스플리팅
- Protected routes (AuthGuard, AdminGuard)

## Backend / Database

### Supabase

- **PostgreSQL**: 관계형 데이터베이스
- **Auth**: PKCE 플로우 기반 인증 (Google, Kakao OAuth + Email/Password)
- **Row-Level Security (RLS)**: 행 수준 보안 정책
- **RPC**: 서버 사이드 함수 호출 (check_user_status 등)

## Styling

- **CSS Custom Properties**: 테마별 변수 시스템
- **Glassmorphism**: 블러 효과 기반 UI
- **5가지 컬러 테마**: Blue, Red, Green, Purple, Orange
- **Dark/Light/Auto 모드**: 시스템 설정 연동 가능

## 폰트

- **Noto Sans KR**: Google Fonts (weights 300-800)
- **Font Awesome 6.5.1**: 아이콘 시스템

## 배포
