# 컴포넌트 가이드

## Layout 컴포넌트

### Navbar

Glassmorphism 효과가 적용된 내비게이션 바.

**파일**: `src/components/layout/Navbar.jsx`, `Navbar.css`

#### 기능

| 기능 | 설명 |
|------|------|
| **Glassmorphism** | `backdrop-filter: blur(12px)` 블러 효과 |
| **Color Picker** | 5가지 컬러 테마 선택 (Blue, Red, Green, Purple, Orange) |
| **Theme Toggle** | Light/Dark/Auto 모드 전환 |
| **Language Toggle** | 한국어/영어 전환 |
| **User Menu** | 로그인/로그아웃, 프로필, 설정 |
| **Mobile Menu** | 햄버거 메뉴, 슬라이드 인 방식 |

#### 구조

```jsx
<nav className="navbar">
  <div className="navbar-container">
    {/* 로고 */}
    <Link to="/" className="navbar-logo">
      <span className="logo-text">OpenClaw</span>
    </Link>

    {/* 네비게이션 링크 */}
    <div className="nav-links">
      <Link to="/intro">소개</Link>
      <Link to="/edu-hub">학습 허브</Link>
      <Link to="/tools">AI 도구</Link>
      <Link to="/resources">학습 자료</Link>
      <Link to="/community/board">커뮤니티</Link>
    </div>

    {/* 유틸리티 영역 */}
    <div className="nav-utils">
      <ColorPicker />
      <ThemeToggle />
      <LanguageToggle />
      <UserMenu />
    </div>

    {/* 모바일 메뉴 버튼 */}
    <button className="mobile-menu-btn">
      <i className="fas fa-bars"></i>
    </button>
  </div>
</nav>
```

## AI 컴포넌트

### TwoColumnLayout

AI 도구의 2컬럼 레이아웃. 좌측은 입력 폼, 우측은 AI 출력.

**파일**: `src/components/ai/TwoColumnLayout.jsx`, `TwoColumnLayout.css`

```jsx
<TwoColumnLayout
  title="강의계획서 생성기"
  leftColumn={<InputForm />}
  rightColumn={<StreamingOutput />}
/>
```

| Props | 타입 | 설명 |
|-------|------|------|
| `title` | string | 도구 제목 |
| `leftColumn` | ReactNode | 왼쪽 영역 (입력 폼) |
| `rightColumn` | ReactNode | 오른쪽 영역 (AI 출력) |

### StreamingOutput

AI 스트리밍 응답을 실시간으로 표시하는 컴포넌트.

**파일**: `src/components/ai/StreamingOutput.jsx`, `StreamingOutput.css`

```jsx
<StreamingOutput
  content={output}
  isStreaming={isStreaming}
  error={error}
/>
```

| Props | 타입 | 설명 |
|-------|------|------|
| `content` | string | Markdown 형식의 출력 내용 |
| `isStreaming` | boolean | 스트리밍 진행 중 여부 |
| `error` | string/null | 에러 메시지 |

- `react-markdown`과 `remark-gfm`을 사용하여 Markdown 렌더링
- 스트리밍 중 커서 애니메이션 표시
- 에러 발생 시 에러 메시지 표시

### ExportToolbar

생성된 콘텐츠를 다양한 형식으로 내보내는 도구모음.

**파일**: `src/components/ai/ExportToolbar.jsx`, `ExportToolbar.css`

```jsx
<ExportToolbar
  content={output}
  title="강의계획서"
  isVisible={!isStreaming && output}
/>
```

| Props | 타입 | 설명 |
|-------|------|------|
| `content` | string | 내보낼 콘텐츠 (Markdown) |
| `title` | string | 파일명에 사용될 제목 |
| `isVisible` | boolean | 도구모음 표시 여부 |

- 5가지 형식: PDF, Markdown, JSON, CSV, HTML

### AIProviderSelect

AI 제공자와 모델을 선택하는 컴포넌트.

**파일**: `src/components/ai/AIProviderSelect.jsx`, `AIProviderSelect.css`

```jsx
<AIProviderSelect
  provider={provider}
  model={model}
  onProviderChange={setProvider}
  onModelChange={setModel}
/>
```

| Props | 타입 | 설명 |
|-------|------|------|
| `provider` | string | 선택된 AI 제공자 |
| `model` | string | 선택된 모델 |
| `onProviderChange` | function | 제공자 변경 콜백 |
| `onModelChange` | function | 모델 변경 콜백 |

## Common 컴포넌트

### HeroCarousel

5개 슬라이드의 자동 회전 캐러셀.

**파일**: `src/components/common/HeroCarousel.jsx`, `HeroCarousel.css`

- **슬라이드 수**: 5개
- **자동 회전**: 5초 간격
- **수동 탐색**: 좌/우 화살표, 하단 도트 인디케이터
- **애니메이션**: 페이드 인/아웃 트랜지션

### HeroBackground

4가지 타입의 애니메이션 배경 효과.

**파일**: `src/components/common/HeroBackground.jsx`, `HeroBackground.css`

| 타입 | 설명 |
|------|------|
| `particles` | 떠다니는 입자 효과 |
| `network` | 네트워크 연결선 효과 |
| `orbs` | 빛나는 구체 효과 |
| `geometric` | 기하학적 도형 효과 |

```jsx
<HeroBackground type="particles" />
```

### AuthGuard

인증된 사용자만 접근 가능하도록 보호하는 래퍼 컴포넌트.

**파일**: `src/components/common/AuthGuard.jsx`

- 이전 위치(location.state.from) 기억

### AdminGuard

## Context Providers

### ThemeProvider

```jsx
// 제공하는 값
{
  theme: 'light' | 'dark' | 'auto',
  colorTheme: 'blue' | 'red' | 'green' | 'purple' | 'orange',
  setTheme: (theme) => void,
  setColorTheme: (color) => void,
}
```

### LanguageProvider

```jsx
// 제공하는 값
{
  language: 'ko' | 'en',
  setLanguage: (lang) => void,
  t: (key) => string,  // 번역 함수
}
```

### AuthProvider

```jsx
// 제공하는 값
{
  user: User | null,
  loading: boolean,
  isAdmin: boolean,
  signIn: (email, password) => Promise,
  signUp: (email, password, name) => Promise,
  signOut: () => Promise,
  signInWithGoogle: () => Promise,
  signInWithKakao: () => Promise,
}
```

### ToastProvider

```jsx
// 제공하는 값
{
  showToast: (message, type) => void,
  // type: 'success' | 'error' | 'warning' | 'info'
}
```
