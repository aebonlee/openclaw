# 내보내기 기능

## 개요

AI 도구에서 생성된 콘텐츠를 5가지 형식으로 내보낼 수 있다. 모든 내보내기 파일에는 "Generated by OpenClaw" 브랜딩이 포함된다.

## 지원 형식

| 형식 | 확장자 | 설명 |
|------|--------|------|
| **PDF** | `.pdf` | jsPDF 기반, 헤더/푸터/페이지네이션 포함 |
| **Markdown** | `.md` | 원본 Markdown 텍스트 |
| **JSON** | `.json` | 메타데이터 포함 구조화 데이터 |
| **CSV** | `.csv` | UTF-8 BOM, 테이블 데이터 추출 |
| **HTML** | `.html` | 스타일 포함 독립 실행 가능 파일 |

## PDF 내보내기

jsPDF와 html2canvas를 사용하여 PDF를 생성한다.

### 주요 기능

- **헤더**: "OpenClaw" 로고 + 문서 제목
- **푸터**: 페이지 번호 (Page X of Y) + 생성 날짜
- **페이지네이션**: 콘텐츠 길이에 따라 자동 페이지 분할
- **한국어 지원**: 한글 폰트 내장

```javascript
import jsPDF from 'jspdf';

  doc.setFontSize(10);

  doc.text(title, 14, 16);

  const imgData = canvas.toDataURL('image/png');

  // 페이지 분할 처리
  const pageHeight = doc.internal.pageSize.getHeight();
  let position = 25; // 헤더 아래부터

  // ... 페이지 분할 로직 ...

  // 푸터
  const totalPages = doc.internal.getNumberOfPages();
  for (let i = 1; i <= totalPages; i++) {
    doc.setPage(i);
    doc.setFontSize(8);
    doc.text(
      `Page ${i} of ${totalPages}`,
      doc.internal.pageSize.getWidth() / 2,
      pageHeight - 10,
      { align: 'center' }
    );
  }

  doc.save(`${title}.pdf`);
  document.body.removeChild(element);
}
```

## Markdown 내보내기

원본 Markdown 텍스트를 `.md` 파일로 저장한다.

```javascript
export function exportToMarkdown(content, title) {
  const header = `# ${title}\n\n> Generated by OpenClaw (https://openclaw.dreamitbiz.com)\n\n---\n\n`;
  const fullContent = header + content;

  const blob = new Blob([fullContent], { type: 'text/markdown;charset=utf-8' });
  downloadBlob(blob, `${title}.md`);
}
```

## JSON 내보내기

메타데이터와 함께 구조화된 JSON 형식으로 내보낸다.

```javascript
export function exportToJSON(content, title, metadata = {}) {
  const data = {
    title,
    content,
    metadata: {
      generator: 'OpenClaw',
      url: 'https://openclaw.dreamitbiz.com',
      generatedAt: new Date().toISOString(),
      ...metadata,
    },
  };

  const blob = new Blob(
    [JSON.stringify(data, null, 2)],
    { type: 'application/json;charset=utf-8' }
  );
  downloadBlob(blob, `${title}.json`);
}
```

## CSV 내보내기

Markdown 콘텐츠에서 테이블 데이터를 추출하여 CSV 형식으로 내보낸다.

### 주요 기능

- **UTF-8 BOM**: 한국어 Excel 호환성을 위해 BOM(Byte Order Mark) 추가
- **테이블 추출**: Markdown 테이블 (`| ... | ... |`) 형식을 파싱하여 CSV로 변환

```javascript
export function exportToCSV(content, title) {
  // UTF-8 BOM 추가 (한국어 Excel 호환)
  const BOM = '\uFEFF';

  // Markdown 테이블 추출
  const tables = extractMarkdownTables(content);

  let csvContent = '';
  tables.forEach((table, index) => {
    if (index > 0) csvContent += '\n\n';
    csvContent += table.headers.map(h => `"${h}"`).join(',') + '\n';
    table.rows.forEach(row => {
      csvContent += row.map(cell => `"${cell}"`).join(',') + '\n';
    });
  });

  const blob = new Blob(
    [BOM + csvContent],
    { type: 'text/csv;charset=utf-8' }
  );
  downloadBlob(blob, `${title}.csv`);
}
```

### Markdown 테이블 추출

```javascript
function extractMarkdownTables(markdown) {
  const tables = [];
  const lines = markdown.split('\n');
  let currentTable = null;

  for (const line of lines) {
    if (line.trim().startsWith('|') && line.trim().endsWith('|')) {
      const cells = line
        .split('|')
        .filter(cell => cell.trim() !== '')
        .map(cell => cell.trim());

      // 구분선 (|---|---|) 건너뛰기
      if (cells.every(cell => /^[-:]+$/.test(cell))) continue;

      if (!currentTable) {
        currentTable = { headers: cells, rows: [] };
      } else {
        currentTable.rows.push(cells);
      }
    } else if (currentTable) {
      tables.push(currentTable);
      currentTable = null;
    }
  }

  if (currentTable) tables.push(currentTable);
  return tables;
}
```

## 공통 다운로드 유틸리티

```javascript
function downloadBlob(blob, filename) {
  const url = URL.createObjectURL(blob);
  const link = document.createElement('a');
  link.href = url;
  link.download = filename;
  document.body.appendChild(link);
  link.click();
  document.body.removeChild(link);
  URL.revokeObjectURL(url);
}
```
