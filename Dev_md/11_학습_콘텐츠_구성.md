## 프롬프트 실습

프롬프트 엔지니어링 기법을 직접 연습할 수 있는 환경.

### 기본 기법

| 기법 | 설명 |
|------|------|
| **역할 설정** | AI에게 특정 역할을 부여 (예: "당신은 교육 전문가입니다") |
| **지시문 작성** | 명확하고 구체적인 지시문 작성법 |
| **출력 형식 지정** | Markdown, JSON, 표 등 원하는 형식 지정 |
| **제약 조건 설정** | 글자 수, 항목 수 등 제약 조건 부여 |

### 고급 기법

| 기법 | 설명 |
|------|------|
| **Chain-of-Thought** | 단계별 사고 과정을 유도하는 프롬프트 |
| **Few-shot Learning** | 예시를 제공하여 패턴을 학습시키는 기법 |
| **구조화 프롬프트** | 체계적으로 구조화된 프롬프트 작성 |
| **반복 개선** | 결과를 기반으로 프롬프트를 반복 수정 |

### 연습 문제

각 기법별 실습 문제를 제공하며, 직접 프롬프트를 작성하고 결과를 확인할 수 있다.
