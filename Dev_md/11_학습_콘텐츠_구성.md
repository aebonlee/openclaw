# 학습 콘텐츠 구성

## 콘텐츠 개요

OpenClaw는 AI 학습 플랫폼으로서 다양한 학습 콘텐츠를 정적 데이터 기반으로 제공한다.

---

## 1. AI 학습자료 (/resources)

### 4개 카테고리, 16개 주제

| 카테고리 | 주제 수 | 내용 |
|----------|---------|------|
| **AI 이론** | 4개 | AI 기초 개념, 머신러닝 원리, 딥러닝 구조, 자연어 처리 |
| **프로그래밍** | 4개 | Python 기초, 데이터 구조, 알고리즘, 라이브러리 활용 |
| **데이터 분석** | 4개 | 데이터 수집, 전처리, 시각화, 통계 분석 |
| **AI 활용** | 4개 | AI 서비스 활용, API 연동, 프로젝트 실습, 윤리와 사회 |

### 프롬프트 연동
- AI 학습자료 사이드바에서 프롬프트 실습/갤러리로 링크
- 드롭다운 3차 메뉴 구조 (ck-nav-group/parent/children)
- 기본 닫힘 상태, 클릭 시 펼침

## 2. 프롬프트 실습 (/prompt-practice)

### 사이드바 5개 섹션

#### 기본 프롬프트 (3개 예시)
- 텍스트 요약 프롬프트
- 코드 리뷰 프롬프트
- 번역 프롬프트

#### 고급 프롬프트 기법 (4개)
- 역할 부여 (Role Playing)
- 단계별 추론 (Chain of Thought)
- Few-shot Learning
- 구조화된 출력 (Structured Output)

#### 실습 챌린지 (3개)
- 챌린지 1: 블로그 포스트 작성
- 챌린지 2: 데이터 분석 보고서
- 챌린지 3: 코드 생성 및 디버깅

#### 퀴즈 (5문항)
- 프롬프트 엔지니어링 관련 객관식 퀴즈
- 정답 확인 및 해설 제공

#### 프롬프트 갤러리 링크
- /prompt-gallery 페이지로 연결

## 7. OpenClaw 강의 (/openclaw-guide)

### 8개 섹션 전문 교육 콘텐츠

| 섹션 | 제목 | 내용 |
