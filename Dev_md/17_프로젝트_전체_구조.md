# 17. 프로젝트 전체 구조

**날짜:** 2026-03-31
**프로젝트:** openclaw (https://openclaw.dreamitbiz.com)

## 기술 스택
- React 19 + Vite 6
- React Router v7
- Supabase (인증, DB)
- jsPDF (PDF 생성)
- react-markdown + remark-gfm
- GitHub Pages 배포

## OpenClawGuide 9개 섹션
1. `#overview` — 학습 개요
2. `#slides` — 학습 자료 PT (PDF 뷰어 + 다운로드)
3. `#architecture` — 시스템 구조
4. `#install` — 설치와 실행
5. `#onboarding` — 온보딩
6. `#channels` — 채널 연결 (Telegram/WhatsApp)
7. `#models` — 모델과 도구
8. `#security` — 보안과 권한
9. `#operations` — 운영과 점검

## 빌드 & 배포
```bash
npx vite build        # dist/ 생성
npx gh-pages -d dist  # GitHub Pages 배포
```
