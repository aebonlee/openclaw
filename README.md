# OpenClaw - AI 학습 플랫폼

AI와 프로그래밍을 배우는 종합 학습 플랫폼

## 주요 기능

- 6개 학습경로 (AI 기초, 프롬프트 엔지니어링, 웹 개발, 데이터 사이언스, Python, AI 도구 활용)
- 42+ 학습 콘텐츠 (7개 카테고리)
- AI 역량 체크리스트
- 프롬프트 엔지니어링 실습
- 학습 자료 라이브러리
- 커뮤니티 게시판
- 한국어/영어 다국어 지원
- 5가지 컬러 테마 + 다크/라이트 모드

## 기술 스택

React 19 | Vite 6 | Supabase | react-router-dom 7

## 실행

```bash
npm install
npm run dev
```
